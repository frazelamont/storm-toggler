Storm Toggler

Toggle UI states using javascript.

What?

  • Toggle documentElement classes based on target, style agnostic so you can apply your own styles and transitions
  • Uses id/href-based targeting, so falls back to a fragment identifier link
  • Adds and toggles ARIA attributes
  • Built-in transition delays and animating-out class to hook transitions onto, avoiding unwanted animations on load/resize
  • Based on standard HTML5 markup

Useful for:

  • off-canvas elements, including hamburger or kebab navigation
  • modals (excluding gallery modals)
  • dropdown toggles
  • show/hide

Installation

npm install storm-toggler

Usage

var StormToggler = require('storm.toggler');StormToggler.init(d.querySelectorAll('.js-toggle'));

Options

delay, Number, milliseconds the animating class persists, corresponding to duration of your CSS animation/transition, default 200ms

var StormToggler = require('storm.toggler');StormToggler.init(d.querySelectorAll('.js-toggle', {'delay': 360}));

targetLocal, boolean, to determine where the state classes are added so the toggler can toggle global (page-wide) state or local (subcollection of DOM), default false

var StormToggler = require('storm.toggler');StormToggler.init(d.querySelectorAll('.js-toggle', {'targetLocal': true}));

Off-canvas menu, as seen on this page

HTML

The href (or data-target for a none-anchor element) of the trigger button must match the id of the target elemenbt. In the event of JS failure any trigger anchors serve as fragment identifier links.

<a href="#navigation" class="hamburger js-toggle" title="Navigation menu" aria-label="Navigation menu">   <svg fill="#000000" height="24" viewBox="0 0 24 24" width="24" xmlns="http://www.w3.org/2000/svg">      <path d="M0 0h24v24H0z" fill="none"/>       <path d="M3 18h18v-2H3v2zm0-5h18v-2H3v2zm0-7v2h18V6H3z"/>   </svg></a>
<nav role="navigation" id="navigation" class="nav-primary">   <a href="#" class="nav-primary-item nav-primary-item--current">Home   <a class="nav-primary-item" href="#">About</a>   <a class="nav-primary-item" href="#">Contact</a></nav>

CSS

The CSS hooks for animation are '.on--' + the id of the target element, and 'animating--' + the id of the target element.

.nav-primary {   position: fixed;   top:0;   left:0;   width:50%;   bottom:0;   padding-top:10%;   visibility: hidden;   background:#191919;   transform: translateX(-100%);}main {   transform: translateX(0%);   padding:10% 10%;   background:#fff;}.on--navigation .nav-primary {   visibility: visible;   transition: transform 260ms;   transform: translateX(0);}.on--navigation main {   transition: transform 260ms;   transform: translateX(50%);}.on--navigation svg {   fill:#fff;}.animating--navigation .nav-primary {   transform: translateX(-100%);   transition:transform 260ms ease-out;}.animating--navigation main {   transition: transform 260ms;   transform: translateX(0%);}
